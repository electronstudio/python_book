Print Syntax

  1. Write a program that outputs Helloworld.

  2. Which of these programs is correct?

    1. print(hello world)

    2. print(hello_world)

    3. print("hello world")

    4. print "hello world)

    5. print hello_world

  3. Which of these programs is correct?

    1. print{"hello world"}

    2. print["hello_world"]

    3. print("hello world")

    4. print "hello world"

    5. print <"hello_world">

  4. This incorrect program has a syntax error. Write a correct version with no error.

    print("hello world)

  5. This incorrect program has a syntax error. Write a correct version with no error.

    1. print(“Dinner is ready”)

    2. output(“Dinner is ready”)

    3. say(“Dinner is ready”)
